Add glitch-waveform component for audio visualization

New `<glitch-waveform>` component that visualizes audio frequency data with:

- Configurable number of bars with flexible width support
- Ghost peak indicators with decay effect
- Chromatic aberration on hover
- Real-time AnalyserNode connection for frequency data
- Idle animation when no audio source connected
- CSS custom properties for styling (--bar-flex, --bar-width, --bar-gap)
- Reduced motion support
