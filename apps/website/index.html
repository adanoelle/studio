Glitch Text Component

Hover over text to see it glitch. Embodies the concept of unstable, multiple identities.

Mode: All

Glitches every character randomly:

<glitch-text text="..." mode="all">

Mode: Word

Glitches one random word at a time:

<glitch-text text="..." mode="word">

Mode: Character (Vowels)

Only glitches vowels:

<glitch-text mode="character" character-type="vowels">

Intensity Variation

High intensity (0.8):

<glitch-text intensity="0.8">

Glitch Border Component

Hover over borders to see them break. Represents refusing containment.

Overworld + Shift

Cool digital aesthetic with shifting borders.

break-pattern="shift"

Overworld + Gap

Borders develop gaps when glitching.

break-pattern="gap"

Underworld + Color

Warm organic aesthetic with color shifts.

break-pattern="color"

Underworld + All

All glitch effects combined.

break-pattern="all"

Theoretical Framework

This design system embodies concepts from feminist theory:

Legacy Russell: Glitch Feminism

  • - breaking expected patterns
  • - chromatic aberration
  • - intentional imperfection

Maria Lugones: Worldtraveling

  • - public vs personal spaces
  • - explicit transitions
  • - refusing single identity
