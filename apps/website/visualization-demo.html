How It Works

This demo uses Hydra - a live coding video synth by Olivia Jack. Hydra runs on GPU shaders and creates the psychedelic, reactive visuals common in algorave and livecoding performances.

  • bass(), mid(), treble() - frequency bands drive oscillators
  • beat() - pulses on each pattern event (hap)
  • Feedback loops - create trails and echoes
  • Kaleidoscope - geometric symmetry from audio
  • Modulation - layers blend based on audio energy
// Hydra code (runs on GPU) osc(10, 0.1, () => bass() * 2) .color(1.2, 0.4, 0.8) .kaleid(() => 3 + Math.floor(treble() * 6)) .modulate(noise(3), () => mid() * 0.3) .out()
